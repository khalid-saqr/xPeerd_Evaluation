# Locked TRACE-R methods

The package retains the validated TRACE-R v3 analytical approach: deterministic concern-unit extraction; TF–IDF and token-overlap attested-alignment proxies; unique bipartite cross-source concern matching; category coverage; explicit targeting, reasoning, executability and relevance measures; recommendation normalization; Pearson, Spearman, Kendall, distance correlation, partial rank correlation, Lin concordance, MAE, RMSE, Bland–Altman diagnostics, bootstrap intervals, permutation tests, Wilcoxon tests, exact McNemar tests, quadratic weighted kappa and Benjamini–Hochberg correction.

The strict complete-case rule is applied before analysis. No missing xPeerd reviewer report is imputed or reconstructed.
